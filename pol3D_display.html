introduction to the pol3D Function Library for WebGL

I got to know the WebGL language occasionally when I was browsing internet. WebGL is a javascript based language that provide interface to get access to GPU for HTML5 website. It enriches the content of web page with 3D Animation, as well as provide high efficiency of graphic calculation because part of the code is compiled and send to GPU to execute. I tried to integrate the language into a library which I call “pol3D” of functions and I regard the construction of this library as the first true exploration in the world of Computer Graphics because I did most of the thing at the level of bottom and from the level of bottom, which makes me understood many details of 3D CG. The library is aimed for using least code to create a 3D world on the web page, and provide a friendly interface for user. It contains 9 “.js” files including “polBackground.js”, “polCamera.js”, “polFileReader.js” and so on, each aimed for an aspect of the whole work so that if one is not interested in some aspects, he can make his code lighter by not including the corresponding file. During the process of the programming, I learned the construction of projection translation matrix, technique of “Pick Up” and so on. I even defined a new type of file of “.pol” that contains the vertex information of an object and a converter that convert a file of “.obj” to “.pol” for the efficiency of my code. But I have to confess that it is not completed by now and I plan to go on with this work during my winter holiday.
